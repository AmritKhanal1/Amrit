About

ShikshaVerse is an educational platform designed to help students access study materials, notes, and past papers in a simple and organized way. The platform focuses on user-friendly design and accessibility for all students.

Features

Access to university notes and resources

Old exam papers categorized by semester

Responsive and mobile-friendly design

Easy navigation with clean UI

Modern social media integration

Technologies Used

HTML5

CSS3

JavaScript

Google Fonts

Font Awesome (or other icon libraries)

Installation

Clone the repository:

git clone https://github.com/yourusername/shikshaverse.git


Navigate to the project folder:

cd shikshaverse


Open index.html in your preferred browser to view the website.

Usage

Navigate through different sections like Notes, Old Papers, and Resources

Access social media links in the footer

Fully responsive on both desktop and mobile
